COMP 3104 - DevOps Course Overview

The COMP 3104 DevOps course at George Brown College introduces students to the core principles and practices of DevOps, which include automation, continuous integration, continuous delivery, and collaboration between software development and IT operations.

Course Objectives

Understand the DevOps culture and its importance in modern software development.
Learn about the tools and technologies used in DevOps, including Git, CI/CD, Docker, and cloud services.
Gain hands-on experience with version control systems, automation scripts, and deployment pipelines.
Skills Developed

Implementing CI/CD pipelines to automate build, test, and deployment processes.
Using version control tools like Git for effective collaboration.
Configuring infrastructure using automation tools to improve efficiency and reliability.
